### 🚀 启动步骤

#### Step 1: 环境准备 (0.5小时)
```bash
# 后端
cd Qingyu_backend
go mod download
go build -o main .

# 前端
cd Qingyu_fronted
npm install
npm run dev
```

#### Step 2: Docker启动 (0.5小时)
```bash
# 启动所有服务
docker-compose -f docker/docker-compose.yml up -d

# 验证服务
docker ps
```

#### Step 3: 数据库初始化 (0.5小时)
```bash
# 执行初始化脚本
cd Qingyu_backend/migration
mongo init_mongodb.js
```

#### Step 4: API验证 (1小时)
```bash
# 验证后端API
curl http://localhost:8080/api/v1/health

# 验证前端
http://localhost:5173
```

## 📞 问题解决

### 常见问题

**Q: 后端API未响应?**
```
A: 检查步骤：
1. 确认MongoDB连接: docker ps | grep mongo
2. 查看后端日志: tail -f logs/qingyu.log
3. 验证API端点: curl http://localhost:8080/api/v1/health
```

**Q: 前端页面加载缓慢?**
```
A: 检查步骤：
1. 清除浏览器缓存: Ctrl+Shift+Delete
2. 检查网络: 浏览器DevTools → Network
3. 检查构建: npm run build --report
```

**Q: 数据库出错?**
```
A: 恢复步骤：
1. 重初始化数据库
2. 运行迁移脚本
3. 导入初始数据
```
