# UI错乱和页面跳转修复任务背景

## 任务时间
2026-01-25

## 任务描述
用户报告主页出现许多UI错乱问题，且许多页面无法正确跳转。

## 相关背景
- 项目刚刚完成自动路由(unplugin-vue-router)迁移
- 76个页面文件，16个模块已迁移
- 使用约定式路由系统，文件扩展名为`.page.vue`
- 路由命名规范：`module:page`格式

## 需要调查的问题
1. 主页UI错乱
2. 页面跳转失败

## 可能的原因
- 路由迁移后，某些链接仍使用旧的路径
- 组件引用路径可能不正确
- 布局系统可能有问题
- 样式可能丢失

## 调试计划
1. 检查主页组件和路由
2. 检查页面跳转逻辑
3. 检查布局系统
4. 检查样式加载
