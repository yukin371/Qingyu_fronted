## 使用建议

### 图标大小选择

| 场景 | 推荐尺寸 | 图标示例 |
|------|---------|---------|
| 紧凑型UI（标签、徽章） | 12px (xs) | `Bookmark`, `Check` |
| 默认尺寸（按钮、输入框） | 16px (sm) | `Search`, `Edit` |
| 次要操作（导航） | 20px (md) | `ArrowRight`, `Menu` |
| 主要操作（特色内容） | 24px (lg) | `BookOpen`, `Robot` |
| 英雄区域（大展示） | 32px (xl) | `Sparkles`, `Trophy` |
| 空状态（大展示） | 48px (2xl) | `BookNotFound`, `Empty` |

### 图标颜色选择

| 场景 | 推荐颜色 | CSS类名 |
|------|---------|---------|
| 继承文字颜色 | currentColor | `text-current` |
| 主要操作 | cyan-600 | `text-cyan-600` |
| 成功状态 | green-600 | `text-green-600` |
| 警告状态 | yellow-600 | `text-yellow-600` |
| 危险状态 | red-600 | `text-red-600` |
| 次要/禁用 | gray-400 | `text-gray-400` |

### 按场景推荐图标组合

#### 书城页面
```vue
<!-- 搜索栏 -->
<QyIcon name="Search" :size="20" />

<!-- 收藏按钮 -->
<QyIcon name="Bookmark" :size="18" />
<QyIcon name="BookmarkFilled" :size="18" />

<!-- 阅读状态 -->
<QyIcon name="BookOpen" :size="24" />
<QyIcon name="Reading" :size="20" />

<!-- 书架视图 -->
<QyIcon name="Bookshelf" :size="32" />
<QyIcon name="Library" :size="48" />
```

#### 空状态页面
```vue
<!-- 图书未找到 -->
<QyIcon name="BookNotFound" :size="48" class="text-gray-400" />

<!-- 暂无数据 -->
<QyIcon name="NoData" :size="48" class="text-gray-400" />

<!-- 空文件夹 -->
<QyIcon name="EmptyFolder" :size="48" class="text-gray-400" />
```

#### AI功能页面
```vue
<!-- AI助手 -->
<QyIcon name="Robot" :size="32" />

<!-- 语音输入 -->
<QyIcon name="VoiceInput" :size="24" />

<!-- 智能推荐 -->
<QyIcon name="Lightbulb" :size="24" />
<QyIcon name="Sparkles" :size="20" />

<!-- AI生成 -->
<QyIcon name="Brain" :size="32" />
<QyIcon name="NeuralNetwork" :size="48" />
```

## 添加新图标

如需添加新图标，请按以下步骤操作：

1. 从 [Heroicons](https://heroicons.com/) 或 [Tabler Icons](https://tabler-icons.io/) 获取SVG代码
2. 在 `src/design-system/assets/icons.ts` 中添加图标
3. 在 `src/design-system/components/basic/QyIcon/types.ts` 中添加类型定义
4. 更新本文档
